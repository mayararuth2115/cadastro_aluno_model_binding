# cadastro_aluno_model_binding

Atividade da disciplina de Desenvolvimeto Web Utilizando ASP.NET Core MVC

O projeto implementa um formulário de cadastro de aluno com validação de dados e página de confirmação 

## Funcionalidades

Cadastro de Aluno 
Validação de campos obrigatórios 
Validação de e-mail
Página de confirmação do cadastro
